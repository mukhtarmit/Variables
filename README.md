# JavaScript Fundamentals Assignments

This repository contains my JavaScript fundamentals assignments completed as part of my web development learning journey. Each assignment demonstrates a core JavaScript concept using HTML and JavaScript.

## 📂 Assignments

### 1. User Input & Variables
**File:** `user-input-variables.html`

**Description:**
- Accepts user input using `prompt()`.
- Stores values in variables.
- Displays the entered information on the webpage.

**Concepts Covered:**
- Variables
- User Input
- Output using `document.write()`

---

### 2. Students Grade Calculator
**File:** `student-grade-calculator.html`

**Description:**
- Accepts a student's name and score.
- Calculates the corresponding grade using conditional statements.
- Displays the student's result.

**Grading System**
| Score | Grade |
|--------|-------|
| 70–100 | A |
| 60–69 | B |
| 50–59 | C |
| 45–49 | D |
| 40–44 | E |
| Below 40 | F |

**Concepts Covered:**
- Variables
- Conditional Statements (`if...else`)
- User Input
- Output

---

### 3. Introduction to Functions
**File:** `functions.html`

**Description:**
- Demonstrates how JavaScript functions work.
- Accepts a user's name and returns a personalized greeting.

**Concepts Covered:**
- Functions
- Parameters
- Return Statement
- Function Calls

---

### 4. Introduction to Arrays (Advanced)
**File:** `advanced-arrays.html`

**Description:**
This assignment demonstrates advanced introductory array operations including:
- Creating arrays
- Adding elements
- Removing elements
- Sorting arrays
- Reversing arrays
- Searching arrays
- Looping through arrays
- Counting array elements

**Array Methods Used**
- `push()`
- `pop()`
- `sort()`
- `reverse()`
- `includes()`
- `length`

---
